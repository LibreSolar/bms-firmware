Simulator

Zephyr allows to run an application on a Linux or Mac hosts using the native_sim board.

The BMS firmware currently does not mock an entire BMS IC. However, the simulated application allows to test the ThingSet communication and other basic features.

Execute the following command to run the simulated application:

west build -b native_sim -t run

In order to use a Linux host’s Bluetooth adapter (here: hci0) for Zephyr, install BlueZ utils and run the following commands:

west build -b native_sim
sudo btmgmt power off
sudo build/zephyr/zephyr.exe --bt-dev=hci0
